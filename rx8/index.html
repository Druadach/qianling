RX 8 音频修复指南

发表于 2021-06-17 更新于 2021-06-20 分类于 视频制作 阅读次数 评论数

教程

B站首套全40集RX8音频修复中文教程！

驱动

有独立声卡：ASIO，延迟低
无独立声卡：RX Monitor，优于 MME
Buffer size：数字越小，延迟越小，性能要求越高

布局

左下方可调整显示波形、频谱的比例。
底部最右侧工具为包络线。
左上方箭头可以合并显示左右声道。

快捷键

Ctrl + E 导出
Shift + S 静音

基础知识

Bit depth（位深度）：一般为 16-bit/24-bit
Dither：位深度变动时在缺失频率填充的噪音类型

模块

环境音匹配：Ambience Match

把 A 场景的背景噪音采样复制到 B 场景中。
Trim [dB] 控制环境音音量。

呼吸声控制：Breath Control

Gain 增益模式适合呼吸声音量均衡的情况。
Target 目标模式对大呼吸声强衰减，小呼吸声弱衰减。

  • Gain/Target level 调整呼吸声衰减量
  • Sensitivity 灵敏度

中置声道提取：Center Extract

通过中声场、边声场提取人声/背景音，效果没有 Music Rebalance 好。
Keep center 保留中声场（人声）。
Keep sides 保留边声场（音乐）。

  • Reduction strength 处理强度
  • Artifact smoothing 平滑度
  • Dry mix 原始声音和处理后声音的混合比

去除漏音：​De-bleed

如录入了耳机中背景音乐的漏音，这时对背景音乐采样，就能修复混入的声音。
先调整处理强度，再用平滑度调整细节。

  • Reduction strength 处理强度
  • Artifact smoothing 平滑度

去除爆音：De-click

去除口水音、鼠标声。
纯人声用 ​​Mouth De-click。
Algorithm 算法：

  • Single-band 单波段模式，适合爆音类型都相近的情况
  • Multi-band (periodic clicks) 多波段定时模式，固定间隔发生的爆音
  • Multi-band (random clicks) 多波段随机模式（常用）
  • Low-latency 低延迟模式，适合爆音不明显的情况

参数：

  • Frequency skew 频段偏斜：口水音偏向高频还是低频
  • Click widening：去除太严重时，补偿嘴巴发声的感觉
  • Sensitivity 灵敏度：探测灵敏度

不全选应用，避免缺频；使用 Instant process 和 Brush，手动选择区间处理。

修复削波：De-clip

修复超过 0 dB 的爆频和削波。

  • Threshold：根据爆音区域选择上或下，参数根据预览情况调整。
  • Post-limiter 限制到 0 dB 内
  • Suggest：不准，建议手动调整

去除低电平高频次爆音：De-crackle

修复短暂的噼啪、点击声、椅子吱呀声。

  • Quality 处理质量：根据电脑性能调整
  • Strength 处理强度：参数根据预览情况调整
  • Amplitude skew 幅度偏斜：噪音在人声中则偏左，在空白区域则偏右

去除嘶声/齿音：De-ess

  • Algorithm 算法：
    • Classic：全频段嘶声衰减，嘶声种类多时用
    • Spectral：特定频率衰减，高频嘶声用，齿音就是高频嘶声（常用）
  • Threshold 阈值：高于阈值的嘶声被衰减
  • Absolute 绝对模式：强压，不自然
  • Cutoff freq 临界频率：指定频率以下不衰减，齿音多时临界频率多降低些 3 kHz-4 kHz
  • Speed：根据嘶声长度、快慢选择，z、c、s 这样短促的选 Fast，zh、ch、sh 这样慢而柔和的选 Slow
  • Spectral shaping：处理深度
  • Spectral tilt：频谱补偿，去掉嘶声后缺频部分填充的噪音类型，一般为白噪音

去嗡嗡声：De-hum

修复设备产生的电流声之类的粉红噪和红噪。

  • Adaptive mode 自适应模式：电流声有变化时勾选
  • Frequency：衰减频率位置
  • Filter Q：衰减量，值越小衰减量越大
  • HP filter 高通滤波器：低切，作用不大
  • LP filter 低通滤波器：高切，作用不大
  • Filter DC offset：所有嗡嗡声都衰减
  • Linear-phase filters：提升衰减精度
  • Harmonics 谐波：谐波次数越多，衰减强度越强
  • Link harmonics：All 以基频点为中心统一调整；Odd/even，Slope odd 奇次谐波、Slope even 偶次谐波、None 自由调整
  • Slope：泛音衰减强度，减少人声受损程度

去除喷麦：De-plosive

修复瞬间气流过大引起的爆破音。

  • Sensitivity 灵敏度：过高会破坏声音低频
  • Strength 处理强度：过高会破坏声音低频
  • Frequency limit 频率限制：指定频率以下的频率将被探测到，男声 200 Hz，女声 350 Hz

处理后，再用 EQ 提升指定频率以下的低频。

去除混响：De-reverb

处理质量没有 Dialogue De-reverb 高，但处理深度好，处理得干净。
在预览中调整各参数。

  • Reduction 衰减强度
  • Reverb Profile：混响在各频率占比，哪儿多提升哪儿
  • Tail length 尾长：每段混响的持续时间
  • Artifact smoothing 平滑度：去除混响的准确率、强度
  • Enhance dry signal：干信号增强

去除摩擦声：De-rustle

针对桌、椅、门、衣服等摩擦声。

  • Reduction Strength 衰减强度
  • Ambience preservation 背景声保护：数值越低背景声保留越少，摩擦声去除越多
  • Separation algorithm 分离算法：
    • Channel independent：普通质量处理模式
    • Joint channel：适合立体声文件，且左右声道一致
    • Advanced joint channel：高质量处理模式，配置要求高

去除风声：​De-wind

在预览中调整各参数。

  • Reduction 衰减强度
  • Crossover frequency 频率限制：指定频率以下的风声才被去除
  • Fundamental recovery：谐波恢复
  • Artifact smoothing 平滑度：影响过渡时的自然效果

信噪分离：Deconstruct

电吉他刺耳、失真、爆频噪音，将干音分离成噪音和有用信号来调节比例，还可进一步调节瞬态。

  • Tonal gain 音乐增益
  • Noisy gain 噪音增益
  • Transient gain 瞬态增益：勾选 Separate transients 后启用，增强音乐冲击感
  • Tonal/noisy balance 音乐/噪音平衡：偏向音乐还是噪音
  • Artifact smoothing 平滑度

若从带人声的背景音乐中提取对话，参数：

  • Tonal gain：-inf.
  • Noisy gain：0
  • Tonal/noisy balance：+2.0
  • Artifact smoothing：3.3

对话语调调整：Dialogue Contour

​调节局部人声的包络、音高、共振峰。

  • Curve smoothing 平滑度：影响过渡时的自然效果
  • Formant scaling：共振调整，影响音色
  • Pitch offset：整体语调提升或衰减

对话去混响：Dialogue De-reverb

  • Reduction 衰减强度
  • Sensitivity 灵敏度
  • Ambience preservation 背景声保护：数值越低背景声保留越少，混响去除越多，真实度下降
  • Separation algorithm 分离算法：
    • Channel independent：普通质量处理模式
    • Joint channel：适合立体声文件，且左右声道一致
    • Advanced joint channel：高质量处理模式，配置要求高

对话噪音分离：Dialogue Isolate

调整人声和背景噪音的比例。

  • Dialogue gain：对话音量
  • Noise gain：噪音音量
  • Dialogue separation sensitivity：左端处理程度最强，右端最弱，一般 5.0
  • Ambience preservation 背景声保护：数值越低背景声保留越少，无背景音真实感会下降
  • Separation algorithm 分离算法：
    • Channel independent：普通质量处理模式
    • Joint channel：适合立体声文件，且左右声道一致
    • Advanced joint channel：高质量处理模式，配置要求高

若从带人声的背景音乐中提取对话使用 Deconstruct。

吉他去除噪音：Guitar De-noise

  • Amp：适合电吉他，去除电流声，需要先选中噪音样本学习
  • Squeak：滑弦噪音，Short 噪音在 200ms 以内，Long 噪音在 1000ms 内
  • Pick：拨片噪音，Attack 建立时间，噪音慢数值小，噪音快数值大

波形修复：Interpolate

修复突然断电的噪音。
对选区附近采样，修复目标区域波形。
框选噪音区域后渲染即可。

  • Quality：处理质量，配置好选最高

去口水音：​​Mouth De-click

  • Sensitivity 灵敏度：探测灵敏度，一般 5 作用
  • Frequency skew 频段偏斜：口水音偏向高频还是低频，可根据光谱判断，一般偏高频，一般 6-8
  • Click widening：去除太严重时，补偿嘴巴发声的感觉

音乐元素分离：Music Rebalance

通过光谱提取人声/背景音。
在预览中调整各参数。

  • Vocal：人声
  • Bass：低音乐器
  • Percussion：打击乐器
  • Other：其他乐器
  • Quality：处理质量，配置好选最高
  • Separation：分离度

光谱降噪：Spectral De-noise

去除拾音底噪。
人声使用 Voice De-noise。
选中底噪，Learn。
没有底噪留白可供学习，勾选 Adaptive mode 自适应模式，Learn time：提前指定秒数计算，时间越长处理越精准。

  • Quality：根据配置选择。
  • Threshold 阈值：越靠上被处理部分越多
  • Reduction 衰减强度：先调整阈值，再调整衰减强度
  • Artifact control：噪音程度控制，值越大处理质量越好
  • Reduction curve：衰减曲线，调整处理不到的频段
  • Smoothing：曲线平滑度

频率再生：Spectral Recovery

录音缺频，只到 15kHz，声音没有高频细节。

  • Learn：全选，学习即可
  • Spectral patching：处理完成后自动填充空白光谱，保持勾选
  • Amount：修复比例，给缺频部分补充多少频率
  • Cutoff：缺频起点
  • Vowel/sibilant balance：声音/嘶声比例
  • Smoothing：平滑度

光谱修复：Spectral Repair

修复无法归类的噪音或解决音频空了一段。能从周围区域采样并且修复选中区域。

Attenuate：选区噪音衰减

  • Bands：插值频段范围
  • Surrounding region length：插值长度
  • Strength：处理强度
  • Multi-resolution：高频噪音则勾选
  • Before/after weighting：偏向前区域采样还是偏向后区域采样
  • Direction of interpolatio：

Replace：在选区附近寻找相似音频替换选区

  • Bands：插值频段范围
  • Surrounding region length：插值长度
  • Multi-resolution：高频噪音则勾选
  • Before/after weighting：偏向前区域采样还是偏向后区域采样

Pattern：在整个音频文件中寻找相似音频替换选区

  • Bands：插值频段范围
  • Multi-resolution：高频噪音则勾选
  • Surrounding region length：插值长度
  • Pattern search range：插值时间范围

Partials + Noise：Replace 的高级版

  • Bands：插值频段范围
  • Multi-resolution：高频噪音则勾选
  • Surrounding region length：插值长度
  • Before/after weighting：偏向前区域采样还是偏向后区域采样
  • Harmonic sensitivity 谐波灵敏度：灵敏度越低检测的谐波越少，声音不自然则减小

人声降噪：Voice De-noise

选中底噪，Learn。
没有底噪留白可供学习，勾选 Adaptive mode 自适应模式。

  • Reduction 衰减强度：先调整阈值，再调整衰减强度
  • Optimize for 优化：
    • Dialogue 对话
    • Music 音乐
  • Filter type 处理风格：
    • Surgical 硬朗
    • Gentle 柔和

Threshold 阈值：调整到刚好盖住原声，再预览根据实际情况调整
Reduction 衰减强度：先调整阈值，再调整衰减强度

磁带失真修复：Wow & Flutter

磁带、黑胶
Wow rate：依据抖动频率选择，应用后波纹被拉直即可

  • Fast：2-5 Hz
  • Medium：0.5-2 Hz
  • Slow：<0.5 Hz

Sensitivity：灵敏度
Center global pitch to：基频调整，一般为 440 Hz

相位调整：Azimuth

相位是波形的前后位置。
左右声道音量不一致、延迟。
选择 Suggest 即可，不用手动调整参数。

  • Level：音量
  • Delay：延迟

统一频响：EQ Match

统一话筒频响、声音，在不同设备、环境补录时更真实。
学习目标频响的音频文件后回到要修改的音频文件，选择 Compare 生成效果。

  • Amount：所选频谱与学习频谱的匹配度

淡出淡入：Fade

Fade in：从前往后拉选区
Fade out：从后往前拉选区

  • Log：对数
  • Linear：线性
  • Cosine：余弦
  • Equal Pwr：等功率

增益：Gain

使用 Instant process，选择区间自动应用。

响度标准化：Leveler

提升整体音量而不爆频。

  • Optimize for 优化：
    • Dialogue：对话
    • Music：音乐
  • Target level：增益量
  • Responsiveness
  • Preserve dynamics
  • Ess reduction：嘶声衰减
  • Breath control：齿音控制

变调、变音：Variable Pitch

  • Curve smoothing：曲线平滑度
  • Preserve time 维持时间：
    • Pitch coherence 音调一致性：过高会生硬
    • Transient sensitivity 瞬态灵敏度：调整动态感

变速：Variable Time

  • Curve smoothing：曲线平滑度
  • Preserve time 维持时间：
    • Pitch coherence 音调一致性：过高会生硬
    • Transient sensitivity 瞬态灵敏度：调整动态感

相同问题查找：Find Similar Event

Similarity：1 为完全相同

声道：Mixing

左右声道合并为单声道：50, 50, 50, 50
右声道变为单声道：0, 100, 0, 100
左声道变为单声道：100, 0, 100, 0

EQ

Analog 模拟：有谐波失真感觉
Digital 数字

  • HP 高通滤波器：低切
  • LP 低通滤波器：高切

相位：Phase

点击 Suggest 调整相位。

修复助手：Repair Assistant

选择合适方案后调整强度。

时间和音调：Time & Pitch

  • Stretch ratio 拉伸率
  • Pitch shift 音调变化

位深度：Dither

响度控制器：Loudness Control

转换采样率：Resample
