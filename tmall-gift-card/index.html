天猫超市享淘卡自动绑定程序

发表于 2024-07-22 更新于 2024-08-01 分类于 编程 Waline： 阅读次数：

1. 功能

输入卡密，批量自动绑定享淘卡，禁止用于撞库等违法行为。

2. DEMO

3. 发布页

猫超卡自动绑定-Quicker

4. 逻辑

4.1 变量

  • card：存储卡信息的变量。
  • isLoggedIn：布尔变量，判断账号是否登录。
  • cardAmount：存储卡数量。
  • tabId：标识浏览器标签页。
  • cardInfo：存储单张卡的卡号和密码。
  • cardNum：存储单张卡的卡号。
  • cardPwd：存储单张卡的卡密码。
  • bindResult：存储绑定操作结果。

4.1.1 步骤：

  1. 打开绑定页：打开绑定页URL https://b2b.tmall.com/card/cardInfo/index?tab=bindingHistory，等待页面加载完成；
  2. 检查登录状态：通过检查登录iframe是否存在判断是否已经登录淘宝账户；
  3. 登录提示：如果用户没有登录，则提示并等待用户登录，若已登录则继续执行代码；
  4. 点击绑定按钮：点击“绑定卡”按钮；
  5. 用户输入：提示用户以指定格式输入卡号和密码；
  6. 读取卡数量：从列表中读取卡数量；
  7. 重复过程：对每张卡重复绑定过程：
    • 读取卡信息；
    • 提取卡号和密码；
    • 在表单中输入卡号和密码；
    • 点击“确认绑定”按钮；
    • 读取绑定结果，如果为“卡密错误”则中断代码并发送通知，如果为“卡已绑定”或“绑定成功”则继续。
  8. 卡全部绑定完成后发送通知。
