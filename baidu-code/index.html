百度云自定义分享提取码

发表于 2017-07-28 分类于 软件技巧 Waline： 阅读次数：

百度云是百度公司推出的一款云服务产品。使用百度云可以轻松地与好友分享你的文件。百度云有一个私密分享功能，即一个百度云分享链接对应一个提取码，好友正确输入提取码之后才能提取文件。

这里介绍几种方法，自定义自己分享出去的文件的提取码。

JavaScript法

Step1 分享文件

在浏览器中打开百度云盘，选中需要分享的文件，然后点击分享按钮；

Step2 修改参数

点击分享按钮后会弹出一个模态框，先不管它，按 F12 打开开发者工具，切换至控制台（Console），将以下代码复制粘贴到控制台，然后回车：

Step3 创建私密链接

点击创建私密链接，会弹出输入框，这时输入你想自定义的密码即可

本节参考自 徐哈哈


油猴插件法

Step1 下载与安装浏览器

如Chrome浏览器、360极速浏览器、360安全浏览器、QQ浏览器、UC浏览器、猎豹浏览器、搜狗浏览器等

Step2 安装Tampermonkey插件

下载并安装Tampermonkey插件，该插件是用来管理浏览器脚本的。
Chrome应用商店链接如下（需要梯子）：tampermonkey

点击右上角蓝色“添加至CHROME”。

  • 如果是非chrome浏览器，下载后直接将crx文件拖入你的浏览器。
  • 如果是chrome浏览器，请将crx文件解压到文件夹中，然后以开发者模式加载。缺点是没法同步。

稍等片刻后会有一个弹窗，点击“添加扩展程序”即可。
至此，插件的下载与安装完成。

点击绿色“安装此脚本”完成安装即可。

Step4 分享文件

打开自己的网盘，分享文件，点击创建私密链接

因为Request被阻止

百度云会一直提示正在创建分享链接，请稍候

注意事项

提取码为任意的4位字符，但不能包含双字节字符,一个字母或数字的字符数是1，一个汉字的字符数是3
