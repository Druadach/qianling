字体子集化

发表于 2017-12-30 更新于 2018-01-07 分类于 前端 Waline： 阅读次数：

Fontmin 和 FontSpider是两款优秀的字体子集化工具，可是 FontSpider 在 Hexo 里使用很不方便，于是，在 Windows 下，采用了 Fontmin 和 FontCreator 搭配的方案。

应用场景

有时候，我们想在网站的 Logo 、Slogan 、标题、活动页中使用自定义字体，通常使用 @font-face 引入 Web 字体，但是完整的中文字体库都是数兆以上的，加载性能非常差，所以我们选择提取部分使用到的字体，这样可以把字体文件变成几KB的大小。


功能

Fontmin 能够从完整的字体库中提取字体并生成 tff 、eot 、woff 、svg 四种格式的文件及它们的 css 引入文件，将网页中需要的字符抽取并集合为一个字体文件，实现按需加载，提升网页文字体验

使用


Fontmin 无需安装，下载即用，但仅支持 .tff 文件

若字体仅发布 .otf 版本，需使用 FontCreator 将字体转化为 .tff 格式（打开-另存为）


下载

Fontmin

FontCreator

FontCreator 仅建议使用 5.6 版，更新的版本无法正常存储字体
