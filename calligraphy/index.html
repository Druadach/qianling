《中国书法创作实践》笔记

发表于 2020-06-03 分类于 生活 阅读次数 评论数

中国书法创作实践 / 丘新巧. -- 上海 : 上海师范大学, 2019.04
SHNU-1205806840

毛笔字书写姿势
站姿：书写大字
坐姿：书写中等字、小字

头正：保持头部端正，略向前俯眼睛与纸面距离大致保持在 40-50 厘米。
身正：上身挺直、两肩齐平、腰部挺起，不可紧贴桌面或弯腰驼背。
臂开：两臂自然撑开，右手执笔、左手按纸，成均衡之势。
足安：两脚自然放平稳，不要交叉、蜷腿、踮脚尖等。

书写一般大小字体时，最合适的方式为悬腕方式。

空中运笔（手部灵活性）

手部稳定性：书法最基本要求，代表对毛笔的控制力。
稳定性衡量标准：能够书写均匀、平稳的线条。

中锋行笔：书写时笔锋始终处于线条中央。
中锋行笔线条：用力均匀、线条呈圆柱状态。

侧锋行笔：书写时笔锋处于线条一侧。
侧锋行笔线条：线条不均匀。

训练视频：中锋行笔（下）
用不同力量书写均匀的、中锋行进的笔画。
用不同速度书写均匀的、中锋行进的笔画。

手部活动训练：能够连贯、流畅地画出顺时针和逆时针的圆圈、正方向和反方向的 S 线。

毛笔字入纸式：笔毫从哪个角度以何种方式入纸。
露锋入纸式：下笔时，笔锋外露在点画外面的起笔方法；应用范围广：行书、草书、日常书写；
藏锋入纸式：下笔时，笔锋藏于点画内的起笔方法；应用范围窄；审美感受：含蓄、内敛、不露锋芒。

横线露锋入纸注意要点：下笔角度尽量趋于 90°；下笔力量要均匀，避免突然收力、增力。
竖线露锋入纸注意要点：下笔角度要平，接近水平 180 度；下笔力量要均匀。
横线藏锋入纸注意要点：下笔角度欲右先左；下笔力量均匀。
竖线藏锋入纸注意要点：下笔角度欲下先上；下笔力量均匀。

藏锋收笔：收笔时笔锋回到点画内再离开纸面，笔画外不露锋芒。
悬针竖：凡竖画下端出锋的，其锋如针之悬，故称”悬针”。
垂露竖：收笔处如下垂露珠，垂而不落。

横线收笔注意要点：多以藏锋方式收笔，即笔锋回到点画内再离开纸面，笔画外不露锋芒。
竖线收笔注意要点：藏锋方式收笔称为垂露竖，露锋方式收笔称为悬针竖。
