Photoshop 学习笔记

发表于 2021-03-29 更新于 2021-06-21 分类于 软件技巧 Waline： 阅读次数：

Photoshop 侧重于图像合成，不擅长画图。

形式美感、易于识别第一位，寓意第二位。

快捷键

Alt：取消按钮→复位按钮。
自动色阶：Ctrl+Shift+L
合并图层：Ctrl+E
反选：Shift+I
通过选区建立图层：Ctrl+J
编组：Ctrl+G
色阶：Ctrl+L

色彩模式

HSB

以人对颜色的感觉为基础。

颜色源于光：

  • 自发光物，如太阳、霓虹灯、屏幕，颜色一般不变（加色模式）。
  • 物的反射光，看见不发光却有颜色的物体时实际看见的是它反射的光的颜色，如花朵、书籍、衣服，不同光源下有色差（减色模式）。

色相（H）：色名，黑和白无色相，0~360° 色相环。
纯度/饱和度（S）：饱和度越高越鲜艳，0 为黑白，0~100%。
明度（B）：越大越亮，越小越暗，0~100%。

RGB

以发光物体为基础，眼睛从光线接收到的颜色。
光三原色，红、绿、蓝，范围 0~255，加色模式。
白：RGB(255, 255, 255)
黑：RGB(0, 0, 0)
灰：RGB(三个相同数值, 越高灰色越浅)
RGB 混合在一起是白光，照射到物体时，物体颜色的互补色被吸收，剩下两种颜色混合在一起形成颜色。

8 位位深度：0~255。

溢色（非印刷安全色）：RGB 的颜色范围和 CMYK 的颜色范围不同，很多颜色只能被看到，不能被印刷出来，所以 RGB 转 CMYK 会降一个灰阶。保证不溢色：视图→校样设置→工作中的 CMYK。

CMYK

颜料三原色（三基色）：在纯白光照射下颜色为青、品红、黄，本身不发光，靠反光被看见，范围 0~100，减色模式。
因印刷颜料总含杂质，用 CMY(100, 100, 100) 无法得到纯黑，故增加 K 代表黑色。

加色模式与减色模式

加色模式与减色模式是色相混合后对颜色明度的影响。
加色模式：颜色越多明度越高，红色+绿色+蓝色=白色，明度 100，不同颜色的光加在一起形成新的颜色。
减色模式：颜色越多明度越低，品红色+黄色+青色=黑色，明度 0，由于颜料吸收特定波段的光，所以只有不被吸收的部分反射回来，颜色越多吸收的光也越多。
减色混合是颜料吸收了其它颜色后所呈现出特定颜色，相当于从白色中减去其他颜色，如：青色=白色-红色；品红=白色-绿色；黄色=白色-蓝色。

互补色能互相吸收，是色相环中相对的颜色。
!(photoshop/photoshop-1.webp)

颜色是加减色同时作用的结果。
!(photoshop/photoshop-2.webp)
①当RGB白色光照射红色R时，品红M完全吸收绿色G，黄色Y完全吸收蓝色B，只有红色R被反射并被眼睛看到；
②当RGB白色光照射绿色G时，青色C完全吸收红色R，黄色Y完全吸收蓝色B，只有绿色G被反射并被眼睛看到；
③当RGB白色光照射蓝色B时，青色C完全吸收红色R，品红M完全吸收绿色G，只有蓝色B被反射并被眼睛看到；
!(photoshop/photoshop-3.webp)
④当RGB白色光照射青色C时，青色C完全吸收红色R，只有绿色G和蓝色B被反射并被眼睛看到，绿色G和蓝色B加色得到青色C；
⑤当RGB白色光照射品红M时，品红M完全吸收绿色G，只有红色R和蓝色B被反射并被眼睛看到，红色R和蓝色B加色得到品红M；
⑥当RGB白色光照射黄色Y时，黄色Y完全吸收蓝色B，只有红色R和绿色G被反射并被眼睛看到，红色R和绿色G加色得到黄色Y；
⑦当RGB白色光照射黑色CMY时，青色C完全吸收红色R，品红M完全吸收绿色G，黄色Y完全吸收蓝色B，没有光被反射并被眼睛看到。

分辨率

位图基于像素，矢量图基于路径。
像素：构成图像的最小单位，像素大小取决于图像分辨率，一个像素只有一种颜色。
分辨率越大，每单位平方英寸的像素越多，细节越多。
PPI：用于显示器，“像素每英寸”，表示每平方英寸内像素数量，72 ppi=72*72=5184 px。
DPI：用于打印机，“墨点每英寸”。

工具

背后：画边缘光晕，需要透明背景
变暗：用来做背景、底图，所有比前景色亮的颜色变为前景色，比前景色暗的颜色不变。
正片叠底和滤色（色的混合）：正片叠底是减色模式，滤色是加色模式。两个相同图层，滤色模式图像变亮，正片叠底变暗。（色环记忆法：两个颜色混合得到色环上两颜色中间的颜色）
柔光：用来屏蔽中性灰（锐化）。
差值、排除：负片效果，差值重一些，排除淡一些。
色相、饱和度、明度：调节单一属性，如饱和度模式时，只提升或降低颜色至前景色饱和度百分比。颜色模式是用来给黑白照片着色的。
颜色：黑白照片着色。
图案图章工具：编辑→定义图案/填充，油漆桶也可填充；新建图层，选择当前图层和下方图层；复杂背景用随机画笔。
历史记录画笔：局部恢复
液化：扭曲、膨胀、收缩。
减淡：提亮。
海绵：调饱和度。
羽化：沿蚂蚁线向内 n 像素淡入，向外 n 像素淡出。
弃用：仿制图章工具、锐化工具、涂抹工具、可选颜色。

通道

颜色通道：用于记录颜色信息，有 RGB 三个通道，每个通道 256 个位置，通道只有黑白灰三色，通道值 0 是黑色，255 是白色，CMYK 相反。
专色通道：补充 CMYK 油墨，如明亮的橙色、绿色、荧光色、金属银色、烫金版、凹凸版、局部光油版。
alpha 通道：用于记录选区，选择→存储选区/载入选区（Ctrl+单击通道），去背景常用。
蒙版：临时通道，黑色遮挡，白色显示。

调色

  • 正确的黑白场
  • 准确的色彩还原
  • 丰富合理的层次
  • 足够的清晰度
  • 适当的饱和度

色阶：直方图，暗部、灰部、亮部；吸管选择最亮和最暗的地方。
曲线：上弦提亮，下弦变暗，S 形加深对比度。
色相/饱和度：模糊选区，Ctrl+U
色彩平衡：中间调是本身颜色，阴影是环境光，高光是光色。新建红色图层，混合模式明度。Alt与下一图层产生剪切关系。
匹配颜色：自动调光、调色；合成两张照片前，先匹配颜色。
通道混合器：高品质灰度图，细节更丰富。
渐变映射。
阴影和高光。
线稿：高反差保留+阈值，擦除黑点。
位图：转灰度后使用，1 位位深度图片，2^1，只有黑白二色；用于报纸印刷，半调网屏，频率 85，用点的疏密产生明暗变化。
索引颜色：合并三个通道为一个通道，减少体积，存储为 Web 所用颜色。

图层

调整图层：常用。
剪贴蒙版：图层内容放入形状。
建立工作路径→剪贴矢量蒙版→当前路径→新建位图蒙版。
锁定透明像素。

滤镜

选择并遮住：粗略勾轮廓线，油漆桶填充要填充区域。
消失点。
像素化→彩色半调：墨点印刷效果，加灰度。
置换。
表面模糊：皮肤变光滑。
镜头模糊：渐变创造通道，选择来源为通道。

流程

  1. 自动色阶：Ctrl+Shift+L
  2. 追加饱和度，图片更鲜艳：Ctrl+U，20~30%
  3. 色彩平衡，强化周围环境，人像偏暖，风景偏冷：调整阴影-5,8,10、高光10,0,20，勾选保持亮度
  4. 锐化：复制为背景，滤镜→其它→高反差保留，1 像素，选择柔光

图片与画布大小匹配：图像→显示全部→裁切
选中区域，按 Alt 取消部分区域
剪贴蒙版：Alt 点击图层
