跳过 Google 开机向导

发表于 2018-08-05 更新于 2019-10-27 分类于 Android 阅读次数 评论数

不同系统刷写 Gapps 后的跳过方法不同，在此做一个总结

1. build.prop 关闭开机引导（推荐）

打开 TWRP 终端

mount -o remount,rw /system　挂载 system 分区为读写

vi /system/build.prop　浏览 build.prop

i　输入 i 进入 Insert 模式

ro.setupwizard.mode=OPTIONAL　找到该行将 OPTIONAL 修改为 DISABLED

ESC　按 ESC 回到 Normal 模式

:wq　输入 :wq 存盘并退出

reboot　重启手机 enjoy it !

2. 首屏四角跳过

从左上角开始，顺时针依次点击四个角（状态栏以下）

3. 路由器代理

在路由器系统中设置网络代理以连接 Google 服务器通过验证

4. 延后安装

刷写系统完毕并开机后再刷入 Gapps，存在 GooglePlay 服务 停止运行的问题

5. 删除验证程序

进入 TWRP，挂载 system，打开 文件管理，删除或重命名 setupwizard.apk
system/priv-app/setupwizard/setupwizard.apk
开机后，挂梯子登录谷歌账户，再把 setupwizard.apk 改回来（前提是不删除源文件，只重命名一下），完成通过向导即可。若未改回会存在任务键、返回键失灵、状态栏无法下拉等问题。
