ModelScope 魔搭社区

两层限额：

  • 账户总限额 2000 次/日；
  • 模型有独立限额，可在此查询 ModelScope API 余量查询。

注意：
deepseek-ai/DeepSeek-V4-Pro 等高级模型在实际调用时根本达不到声明额度，经常提示 insufficient_quota。
我常常只能退而使用 deepseek-ai/DeepSeek-V3.2 和 stepfun-ai/Step-3.5-Flash。
