iOS 旧版 APP 推荐与降级方法（持续更新）

发表于 2021-02-18 更新于 2021-12-20 分类于 iPhone 阅读次数 评论数

前言

　　如果你正在用 2G、3G 运存的 iPhone，一定在为频繁杀后台苦恼叭，不妨改用老版本试试~
　　一般旧版软件的资源占用会更少，因而更流畅、更省电，而且广告也会更少（哭）~
　　甚至某些 App 在旧版本的时候还没开始收费喔！如彩云天气Pro、白描！！
　　所以还不快马住！！
　　交流群：431718023

版本

  • iPhone X 于 2017.11.03 上市，此日期后的版本方支持全面屏；
  • iOS 13.0 Beta 1 于 2019.06.04 上线，此日期后的版本方支持深色模式；
  • iOS 14.5 Beta 1 于 2021.02.02 上线，此日期后的版本一定支持 iOS 14.5 及以后版本；
  • 带 ◆ 为自用版本，在 iPhone SE 2020, iOS 14.3 下运行良好；
  • 谨慎更新系统版本，iOS 14.5 大部分旧版应用无法安装，提示安装包不完整；
  • 各版本更新内容可在七麦数据查询；
  • 如版本失效请留言反馈。

降级

手机端

未越狱

　　使用「爱思极速版」即可安装 App 的主要历史版本，但该应用未在 App Store 上架，安装方法如下：
　　电脑端安装「爱思助手」，USB 连接手机后将自动安装移动客户端「爱思极速版」。
　　Apple Watch 用户勿使用「爱思助手」或「爱思极速版」安装 App，在 Apple Watch 上无法启动，只能 iTunes 抓包或使用 AppStore++ 安装旧版应用（反馈用户：⑨RealZJMM怪我丑（Plex），测试设备：Apple Watch Series1，系统：watchOS 6）。

已越狱

　　安装 AppStore++「应用降级」插件，在 AppStore 中长按「获取」按钮即可降级。

电脑端

　　「爱思极速版」中未收录意向版本时，需使用电脑端抓包。
　　iTunes 下载（v12.6.5.3）：Apple 支持
　　抓包软件下载（发布页：吾爱破解）：蓝奏云

　　使用方法：

  1. 下载并安装旧版 iTunes（v12.6.5.3 内置 App Store）；
  2. 打开 iTunes，在 iTunes 内登录 Apple ID；
  3. 点击由上至下第三行的「♫ 音乐」矩形框，点击「编辑菜单」，勾选「A 应用」，点击「完成」，再次点击「A 应用」完成界面切换；
  4. 下载抓包软件，在程序目录下新建一文件夹，用来存放压缩包内的.exe文件，如：D:\Program Files\iOS旧版应用下载（程序初始化后将在同级目录下生成数据文件，因此特地建立文件夹存放）；
  5. 打开抓包软件，输入 App 名称，按下回车键搜索；
  6. 双击选择要下载的版本；
  7. 点击「开始拦截」按钮；
  8. 打开 iTunes，在右上方「🔍」标识处键入应用名称；
  9. 点击「商店」按钮，选择「在 iTunes Store中搜索“××”」；
  10. 根据设备类型选择「iPhone App」或「iPad App」栏目下的程序，点击「获取」或「下载」按钮开始下载；
  11. 下载完成后，点击由上至下第三行的「资料库」；
  12. 右键应用图标，选择「在 Windows 资源管理器中显示(H)」，即可查看 .ipa 文件；
  13. 将安装包备份到你想存放的位置，以备日后使用；
  14. 下载并安装「爱思助手」；
  15. 打开「爱思助手」，点击左侧列表中的「应用游戏-导入安装」即可安装 .ipa 到设备。

注：
　　软件安装包均与 Apple ID 绑定，如需共享给他人，对方需在 App Store 内登录下载时绑定的 Apple ID，再用电脑导入安装。
　　添加到 iTunes 下载列表后即可停止拦截，抓取下一软件了。

屏蔽应用内更新提示

  1. 越狱；
  2. 安装 AppSync Unified、Filza File Manager；
  3. 打开 Filza File Manager；
  4. 依地址打开 ××× 软件安装目录：/private/var/containers/Bundle/Application/×××/×××.app；
  5. 以文本编辑器方式打开 info.plist；
  6. 将当前版本号一律改为最新版本号，若版本号为 3.1.9，字符串为 3.1.9.1，则改动版本号对应前三位，最后一位不动。

问题

是否应该根据安装包大小选择版本

　　不应该。

反复卸载安装软件后存储空间大幅减少

　　安装「小鸭清理」，反复清理几次「手机空间」即可。

版本过低无法登录

　　安装新版并登录后，选择以下一种方法：
　　(1)手机端：进入「设置-通用-iPhone 存储空间」，点击「卸载 App」后，使用「爱思极速版」安装；
　　(2)电脑端：使用「爱思助手」覆盖安装旧版本。

　　例：QQ v6.2.2 版因版本过低无法登录，则先使用「爱思极速版」安装可登录的 v7.3.5 版，登录后不卸载，使用「爱思助手」覆盖安装 v6.2.2 版；

　　注：
　　尽量在「爱思极速版」中选择版本相近的可登录版本，避免因跨版本数据结构变动过大以致覆盖安装后应用闪退。

iTunes 网络出错

　　下载 0kb：添加下载任务到列表后，停止抓包软件拦截并在 iTunes 内点击「暂停下载」，再点击「继续下载」即可恢复。
　　无法登录：关闭代理软件、抓包软件。
　　若 iTunes 出现其它网络问题，请退出代理软件。

去除「App Store 更新提示」

　　打开 .ipa 文件，删除 iTunesMetadata.plist。

关闭自动更新

　　进入「设置-App Store」，关闭「自动下载：App 更新」。

应用体积比 App Store 安装的大

　　电脑端「爱思助手」安装的应用可进入「设置-通用-iPhone 存储空间」，点击「卸载 App」再点击「重新安装」以缩小体积；「爱思极速版」安装的应用及删除了iTunesMetadata.plist的应用无法缩小体积。

TIM 降级至首版（1.0.4）方法

  1. 旧版 iTunes 抓包如下四个版本：3.1.1、2.5.1、1.1.6、1.0.4；
  2. 使用爱思助手，依次覆盖安装降级：3.1.1→2.5.1→1.1.6→1.0.4；
  3. 每次覆盖后，都需要运行程序一次；
  4. 如遇某版本闪退，重复打开几次就能解决；
  5. 降级至 1.0.4 版本后，进入「设置-通用-iPhone 存储空间」，点击「卸载 App」再点击「重新安装」以缩小应用体积。
