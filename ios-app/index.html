自动获取历史版本 ID

  1. 下载 APP 最新版安装包；
  2. 在抓包软件「iOS旧版应用下载」的「安装管理」中填入安装包所在文件夹地址，找到对应 IPA 安装包，右键选择 查找版本 ID「本地」，即可取得该应用的所有历史版本 ID，版本号由新到旧排序;
  3. 根据点点数据或七麦数据中版本记录自行推断各版本对应版本 ID。

手动获取历史版本 ID

  1. 下载 APP 最新版安装包；
  2. 解压 .ipa 安装包（如压缩软件不支持直接解压则将安装包扩展名改为 .zip）；
  3. 用文本编辑器打开（如记事本）位于安装包根目录的 iTunesMetadata.plist 文件；
  4. 查找 softwareVersionExternalIdentifiers 即可定位至历史版本 ID 所在处，版本号由旧到新排序；
  5. 根据点点数据或七麦数据中的版本记录按顺序自行推断各版本对应的版本 ID。

自动获取版本号及对应历史版本 ID（需越狱）

  1. 越狱；
  2. 安装 AppStore++「应用降级」；
  3. 在 AppStore 中长按「获取」/「更新」按钮，选择「Upgrade/Downgrade」，即可获得全部版本号及版本 ID 条目。

手动获取已安装版本对应版本 ID（需越狱或 TrollStore）

  1. 越狱；
  2. 安装 Filza File Manager；
  3. 打开 Filza File Manager；
  4. 依地址打开 ××× 软件安装目录：/var/containers/Bundle/Application/×××；
  5. 打开 iTunesMetadata.plist；
  6. 展开 Root，倒数第五行的 softwareVersionExternalIdentifier 即为当前安装版本对应版本 ID。

应用体积比正常 App Store 安装的大

进入「设置-通用-iPhone 存储空间」，点击「卸载 App」再点击「重新安装」。

安装提示安装包不完整

系统不兼容，换用系统发布日期以后推出的软件版本。

仅部分应用有效，自行测试。

修改方法

越狱设备

  1. 打开 Filza File Manager；
  2. 依地址打开软件安装目录：/private/var/containers/Bundle/Application/×××；
  3. 搜索 info.plist，打开体积最大项；
  4. 展开 Root，将当前版本号一律改为最新版本号，若版本号为 x.x.x，字符串为 x.x.x.x，则改动版本号对应前三位，最后一位不动。

电脑端

  1. 砸壳（越狱设备或 Mac 可砸壳，无砸壳条件可前往电商平台购买服务，务必甄别是否为完整权限砸壳）；
  2. 解压 .ipa 文件，搜索 info.plist，打开体积最大项；
  3. 将当前版本号一律改为最新版本号，若版本号为 x.x.x，字符串为 x.x.x.x，则改动版本号对应前三位，最后一位不动。

版本过低无法登录

安装新版并登录后，选择以下一种方法：
(1)手机端：进入「设置-通用-iPhone 存储空间」，点击「卸载 App」后，使用「爱思极速版」安装；
(2)电脑端：使用「爱思助手」覆盖安装旧版本。

例：QQ v6.2.2 版因版本过低无法登录，则先使用「爱思极速版」安装可登录的 v7.3.5 版，登录后不卸载，使用「爱思助手」覆盖安装 v6.2.2 版；

注：
尽量在「爱思极速版」中选择版本相近的可登录版本，避免因跨版本数据结构变动过大以致覆盖安装后应用闪退。

iTunes 网络出错

下载 0kb：添加下载任务到列表后，停止抓包软件拦截并在 iTunes 内点击「暂停下载」，再点击「继续下载」即可恢复。
无法登录：关闭代理软件、抓包软件。
若 iTunes 出现其它网络问题，请退出代理软件。

去除「App Store 更新提示」

方法择一：

  1. 使用压缩软件解压缩 .ipa 文件，删除 iTunesMetadata.plist 再重新压缩为 .zip 文件，将扩展名改为 .ipa 即可；
  2. iOS旧版应用下载-安装管理-去App更新提示。

关闭自动更新

  1. 进入「设置-App Store」;
  2. 关闭「自动下载：App 更新」。

版本停止供货

如果抓包版本显示停止供货，注册一个新的 AppleID，登录 iTunes 后的首次抓包可以抓到停供版本。

七、覆盖安装

覆盖安装方法

  • 未越狱系统：打开「爱思助手」，点击左侧列表中的「应用游戏-导入安装」即可覆盖安装 .ipa 到设备；
  • 已越狱系统：使用 Filza 或 AppStore++，即可覆盖安装 .ipa 到设备。

覆盖安装降级案例

  1. 以 TIM 为例，旧版 iTunes 抓包如下四个版本：3.1.1、2.5.1、1.1.6、1.0.4；
  2. 使用爱思助手，依次覆盖安装降级：3.1.1→2.5.1→1.1.6→1.0.4；
  3. 每次覆盖后，都需要运行程序一次；
  4. 如遇某版本闪退，重复打开几次就能解决，若一直闪退则该系统版本不支持。
