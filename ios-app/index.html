降级方法

手机端降级

未越狱系统

  1. 电脑端安装「爱思助手」，USB 连接手机后将自动安装移动客户端「爱思极速版」。
  2. 安装完成后，在手机端的「爱思极速版」内即可安装 App 的主要历史版本。

注：Apple Watch 用户勿使用「爱思助手」或「爱思极速版」安装 App，在 Apple Watch 上无法启动（反馈用户：⑨RealZJMM怪我丑（Plex），测试设备：Apple Watch Series1，系统：watchOS 6）。

版本为 14.0 ~ 15.5 beta 4 的未越狱系统

  1. 安装 TrollStore；
  2. 下载 AppStorePlus-TrollStore（优先）或 DowngradeApp；
  3. 将 AppStorePlus-TrollStore 或 DowngradeApp 的 ipa 文件分享到 TrollStore；
  4. 在 TrollStore 内安装后即可使用 AppStore++ 或 DowngradeApp 降级应用。

已越狱系统

安装 AppStore++「应用降级」插件，在 AppStore 中长按「获取」/「更新」按钮，选择「Upgrade/Downgrade」。

电脑端降级

「爱思极速版」中未收录意向版本时，需使用电脑端抓包。

iTunes 下载（v12.6.5.3）：64 位／32 位

任选一个抓包软件。
抓包软件一：下载地址—蓝奏云（发布页：吾爱破解）（优先）
抓包软件二：下载地址—永硕E盘（发布页：吾爱破解）

使用方法（以抓包软件一为例）：

  1. 下载并安装旧版 iTunes（v12.6.5.3 内置 App Store）；
  2. 打开 iTunes，在 iTunes 内登录 Apple ID；
  3. 点击由上至下第三行的「♫ 音乐」矩形框，点击「编辑菜单」，勾选「A 应用」，点击「完成」，再次点击「A 应用」完成界面切换；
  4. 下载抓包软件，在程序目录下新建一文件夹，用来存放压缩包内的.exe文件，如：D:\Program Files\iOS旧版应用下载（程序初始化后将在同级目录下生成数据文件，因此特地建立文件夹存放）；
  5. 打开抓包软件，输入 App 名称，按下回车键搜索；
  6. 双击选择要下载的版本；
  7. 点击「开始拦截」按钮；
  8. 打开 iTunes，在右上方「🔍」标识处键入应用名称；
  9. 点击「商店」按钮，选择「在 iTunes Store中搜索“××”」；
  10. 根据设备类型选择「iPhone App」或「iPad App」栏目下的程序，点击「获取」或「下载」按钮，选择「Upgrade/Downgrade」；
  11. 下载完成后，点击由上至下第三行的「资料库」；
  12. 右键应用图标，选择「在 Windows 资源管理器中显示(H)」，即可查看 .ipa 文件；
  13. 将安装包备份到你想存放的位置，以备日后使用；
  14. 下载并安装「爱思助手」；
  15. 打开「爱思助手」，点击左侧列表中的「应用游戏-导入安装」即可安装 .ipa 到设备。

注：

  • 添加到 iTunes 下载列表后即可停止拦截，抓取下一软件了。
  • 软件安装包均与 Apple ID 绑定，如需共享给他人，对方需在 App Store 内登录下载时绑定的 Apple ID，再用电脑导入安装。

获取 APP 版本 ID

目前仅存的公开历史版本查询网站「Sunny」已停止版本抓取，仅可查阅 2022 年 06 月 06 日以前的版本 ID，且冷门应用不收录。
如想获取未收录版本 ID 或冷门应用版本 ID 可采用下列方法：

自动获取历史版本 ID

  1. 在 iTunes 中下载 APP 最新版安装包；
  2. 在抓包软件「iOS旧版应用下载」的「安装管理」中填入安装包所在文件夹地址，找到对应 IPA 安装包，右键选择 查找版本 ID「本地」，即可取得该应用的所有历史版本 ID，版本号由新到旧排序;
  3. 根据点点数据中版本记录自行推断各版本对应版本 ID。

手动获取历史版本 ID

  1. 在 iTunes 中下载 APP 最新版安装包；
  2. 解压 .ipa 安装包（如压缩软件不支持直接解压则将安装包扩展名改为 .zip）；
  3. 用文本编辑器打开（如记事本）位于安装包根目录的 iTunesMetadata.plist 文件；
  4. 查找 softwareVersionExternalIdentifiers 即可定位至历史版本 ID 所在处，版本号由旧到新排序；
  5. 根据点点数据中版本记录自行推断各版本对应的版本 ID。

自动获取版本号及对应历史版本 ID（需越狱）

  1. 越狱；
  2. 安装 AppStore++「应用降级」；
  3. 在 AppStore 中长按「获取」/「更新」按钮，选择「Upgrade/Downgrade」，即可获得全部版本号及版本 ID 条目。

获取已安装版本对应版本 ID（需越狱或 TrollStore）

  1. 越狱；
  2. 安装 Filza File Manager；
  3. 打开 Filza File Manager；
  4. 依地址打开 ××× 软件安装目录：/var/containers/Bundle/Application/×××；
  5. 打开 iTunesMetadata.plist；
  6. 展开 Root，倒数第五行的 softwareVersionExternalIdentifier 即为当前安装版本对应版本 ID。

常见问题

安装提示安装包不完整

系统不兼容，换用系统发布日期以后推出的软件版本。

屏蔽应用内更新提示

  1. 越狱；
  2. 安装 AppSync Unified、Filza File Manager；
  3. 打开 Filza File Manager；
  4. 依地址打开 ××× 软件安装目录：/private/var/containers/Bundle/Application/×××；
  5. 搜索 info.plist，打开体积最大项；
  6. 展开 Root，将当前版本号一律改为最新版本号，若版本号为 x.x.x，字符串为 x.x.x.x，则改动版本号对应前三位，最后一位不动。

外区 ID 注册方法

最新&最详细外区AppleID注册教程 - 知乎

反复卸载安装软件后存储空间大幅减少

安装「小鸭清理」，反复清理几次「手机空间」即可。

版本过低无法登录

安装新版并登录后，选择以下一种方法：
(1)手机端：进入「设置-通用-iPhone 存储空间」，点击「卸载 App」后，使用「爱思极速版」安装；
(2)电脑端：使用「爱思助手」覆盖安装旧版本。

例：QQ v6.2.2 版因版本过低无法登录，则先使用「爱思极速版」安装可登录的 v7.3.5 版，登录后不卸载，使用「爱思助手」覆盖安装 v6.2.2 版；

注：
尽量在「爱思极速版」中选择版本相近的可登录版本，避免因跨版本数据结构变动过大以致覆盖安装后应用闪退。

iTunes 网络出错

下载 0kb：添加下载任务到列表后，停止抓包软件拦截并在 iTunes 内点击「暂停下载」，再点击「继续下载」即可恢复。
无法登录：关闭代理软件、抓包软件。
若 iTunes 出现其它网络问题，请退出代理软件。

去除「App Store 更新提示」

方法择一：

  1. 使用压缩软件解压缩 .ipa 文件，删除 iTunesMetadata.plist 再重新压缩为 .zip 文件，将扩展名改为 .ipa 即可；
  2. iOS旧版应用下载-安装管理-去App更新提示。

关闭自动更新

  1. 进入「设置-App Store」;
  2. 关闭「自动下载：App 更新」。

应用体积比 App Store 安装的大

  • 电脑端「爱思助手」安装的应用可进入「设置-通用-iPhone 存储空间」，点击「卸载 App」再点击「重新安装」以缩小体积；
  • 「爱思极速版」安装的应用及删除了 iTunesMetadata.plist 的应用无法缩小体积。

版本停止供货

如果抓包版本显示停止供货，注册一个新的 AppleID，登录 iTunes 后的首次抓包可以抓到停供版本。

覆盖安装

覆盖安装方法

  • 未越狱系统：打开「爱思助手」，点击左侧列表中的「应用游戏-导入安装」即可覆盖安装 .ipa 到设备；
  • 已越狱系统：使用 Filza 或 AppStore++，即可覆盖安装 .ipa 到设备。

覆盖安装降级案例

  1. 以 TIM 为例，旧版 iTunes 抓包如下四个版本：3.1.1、2.5.1、1.1.6、1.0.4；
  2. 使用爱思助手，依次覆盖安装降级：3.1.1→2.5.1→1.1.6→1.0.4；
  3. 每次覆盖后，都需要运行程序一次；
  4. 如遇某版本闪退，重复打开几次就能解决，若一直闪退则该系统版本不支持；
  5. 降级至 1.0.4 版本后，进入「设置-通用-iPhone 存储空间」，点击「卸载 App」再点击「重新安装」以缩小应用体积。
