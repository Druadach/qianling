自用 Windows 软件推荐

发表于 2021-06-06 更新于 2026-03-28 分类于 软件分享 Waline： 阅读次数：

装机必备

  1. 解压缩：7-Zip
  2. 护眼——显示器色温调节：f.lux（UWP）
  3. 系统优化：Dism++
  4. 全局搜索：Listary
  5. 截图、OCR：PixPin
  6. 播放器：PureCodec
  7. 任务栏网速监控：TrafficMonitor（Portable）
  8. 文本编辑器：Visual Studio Code
  9. 自动切换深色/浅色主题：Auto Dark Mode
  10. Proxy：Clash Party

办公

  1. Excel按下Shift横向滚动：OfficeScroll
  2. RPA办公自动化：Quicker
  3. Teams实时字幕导出：Live Captions Saver

杂项

  • ChatGPT 桌面客户端：Chatbox
  • 局域网文件传输：LocalSend
  • 下载：Internet Download Manager
  • 使用时间统计：ManicTime
  • 硬盘占用查看：SpaceSniffer
  • 电子书：Calibre
  • 无边框阅读器：Reader
  • Kindle 笔记管理：糯词笔记、 Clippings Fere
  • 漫画：MangaMeeyaCE
  • 图片压缩：TinyGUI
  • 软件卸载：Geek Uninstaller
  • Gif 录制：ScreenToGif
  • 文件时间修改：NewFileTime
  • 字体子集化：FontMin
  • RSS：irreader
  • 网易云音乐歌词制作：BesLyric
  • 解除占用：IObit Unlocker
  • bilibili 视频下载：哔哩下载姬(跨平台版)
  • 文件预览：QuickLook
  • QQ聊天记录转html：mht2html
  • QQ空间导出：QZoneExport
  • 自动切换壁纸：Yuphiz_AutoTheme
  • 图片加水印：watermarker
  • Switch Pro 手柄连接：BetterJoy
  • 一起看视频

媒体处理

  • 音频处理：Adobe Audition
  • 图形制作：Adobe Illustrator
  • 图像处理：Adobe Photoshop
  • 视频剪辑：Adobe Premiere Pro
  • 视频剪辑：剪映专业版
  • 音频修复：iZotope RX8 Audio Editor
  • 媒体处理：FFmpeg
  • 屏幕录制：Open Broadcaster Software
  • 音频录制：楼月免费MP3录音软件（3.1）
  • 视频压制：MarukoToolbox
  • 视频粗剪：SolveigMM Video Splitter
  • 视频粗剪：SMMVSplitter
  • 视频粗剪：Avidemux
  • 剪映字幕提取：JianYingSRT
  • 字幕制作：Aegisub
  • 字幕制作：ArcTime
  • 图像切分：AvgCutImage
  • 图像拼接：img-handler
  • SQLite数据库编辑：DB Browser for SQLite

开发环境

  • Node.js
  • Git

油猴插件

  • HTML5视频播放器增强脚本
  • 百度网盘简易下载助手（直链下载复活版）
